Simon's Sequence 🎮

About the Game

Test your memory and reflexes with this modern take on a classic game. Watch the neon lights, memorize the sequence, and repeat the pattern. How long can you last before the challenge overwhelms you?

How to Play

  • 👀 Watch the sequence of flashing neon pads.
  • 👉 Repeat the sequence by clicking the pads in the same order.
  • 📈 The sequence gets longer and faster with each level.
  • ❌ One wrong move and the game is over!
